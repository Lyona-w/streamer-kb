# Требования к программному обеспечению

* **Операционная система**: Streamer.bot совместим с 64-разрядными версиями Windows 10 и Windows 11.

Дополнительные компоненты:

* **.NET Framework**: Убедитесь, что установлена последняя версия .NET Framework, необходимая для корректной работы Streamer.bot.
OBS Studio: Для интеграции с OBS Studio необходимо установить соответствующую версию OBS, совместимую с вашей операционной системой.

* **Права доступа**: Убедитесь, что у вас есть административные права на установку и настройку программного обеспечения, а также на управление подключениями к платформам стриминга.
