# Запуск программы

1. Запустите файл Streamer.bot.exe.

2. Если появилось предупреждение от Windows, подтвердите запуск.

3. Дождитесь появления заставки и полной загрузки программы.

## Проверьте служебные папки

1. После первого запуска убедитесь, что рядом с программой создались следующие папки:

* Backup — для резервных копий настроек.

* Data — для данных программы (авторизации, команды, действия и т.д.).

* DLLs — для сторонних библиотек и интеграций.

* Logs — для логов работы программы.

* Updates — для скачанных обновлений.

##  Используйте резервные копии при ошибках

1. Откройте папку Backup, если после изменений программа стала работать неправильно.

2. Восстановите рабочую конфигурацию из сохраненного архива.
